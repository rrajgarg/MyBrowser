# MyBrowser
A browser with following features:
1. Browsing
2. Set Search Engine
3. Private Browsing
4. Simple Browsing
5. Add BookMarks
6. In Simple Browsing your history is automatically updated
7. delete options in history to delete a particular history 
8. check the work of all the tabs at one place
